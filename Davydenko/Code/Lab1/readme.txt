Для запуска программы необходимо открыть приложение для управления базами данных на MySQL,
после чего создать новый sql скрипт, в который вставить содержимое файла delivery_db.sql.

Далее запустить файл delivery.exe и сделать заказ как клиент.

Так же можно запустить adm_dlvr для управления создаными заказами.
