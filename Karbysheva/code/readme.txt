Для запуска программы сначала необходимо скачать установщик с официального сайта и
потом его запустить. После установки приложения выбрать файл (видео, аудио или фото)
одним из способов. Первый способ: через пункт меню "Медиа" в верхней части окна приложения
"Открыть файл" и выбрать необходимый. Второй способ: в приложение через комбинацию клавиш
клавиатуры "Ctrl + O" всплывет окно для выбора файла. Третий способ: через нажатие на файл,
нужный для просмотра, правой кнопкой мыши и выбором пункта "Открыть с помощью" с указанием
необходимого проигрывателя. Четвертый способ: с помощью кнопки "Воспроизведение" в нижней части
окна выбрать файл.
