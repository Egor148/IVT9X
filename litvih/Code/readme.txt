Для сборки программы необходимо скачать и установить дистрибутив с официального сайта. После этого необходимо запустить программу. Далее нужно открыть файл внутри программы:
1 способ: открыть программу, в строке меню выбрать "файл"-"открыть"(Ctrl+O) и выбрать нужный медиафайл(ы) из проводника;
2 способ: открыть нужный медиафайл с помощью программы(правый клик мыши по файлу - открыть с помощью - выбрать из списка программ нужную)
