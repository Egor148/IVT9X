Задание 1. Написать спецификацию в виде сценариев пользователя.

Инварианты: главное окно видимо, программа не закрыта, аудиоустройства подключены.

	Пользовательский сценарий 1. Прослушивание аудиофайла.
Действующее лицо: Пользователь.
Предусловие: программа запущена.
Основной сценарий:
- в строковом меню выбрать «файл» - «открыть…»
- в открывшемся меню проводника найти нужный файл, выделить его и нажать кнопку «открыть»
- на панели управления воспроизведением нажать на кнопку с зеленой стрелкой(Играть)
Постусловие: воспроизводится аудиофайл.
	Пользовательский сценарий 2. Запись звука с микрофона.
Предусловие: программа запущена.
- на панели управления воспроизведением нажать кнопку с красным кругом(Запись)
- поговорить в микрофон некоторое время
- на панели управления воспроизведением нажать на кнопку с черным квадратом(Стоп)
- чтобы прослушать полученные данные, на панели управления воспроизведением нажать на кнопку "Играть" 
Постусловие: записаны аудиоданные.
	Пользовательский сценарий 3. Удаление тишины из аудиофайла.
Предусловие: программа запущена, аудиофайл с тишиной открыт в программе.
- в активном окне программы выделить трек или фрагмент трека для обработки
- в строковом меню выбрать "эффекты" - "обрезка тишины..."
- в открывшемся окне настроить параметры эффекта 
- нажать кнопку "ок" для применения эффекта к аудиоданным
Постусловие: изменены аудиоданные с примененным эффектом.
	Пользовательский сценарий 4. объединение двух файлов в один.
Предусловие: программа запущена, наличие 2-х аудиофайлов.
- открыть первый файл в программу любым удобным способом
- в строковом меню выбрать "файл" - "импорт" - "аудиоданных..."
- в открывшемся меню проводника выбрать второй файл и нажать кнопку «открыть»
- выделить обе аудиодорожки 
- в строковом меню выбрать "треки" - "микшировать" - "свести и обработать"
Постусловие: аудиоданные двух файлов объединены в один.
