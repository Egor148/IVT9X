Задание 1. Написать спецификацию в виде сценариев пользователя.

Инварианты: программа запущена, главное окно видимо, нет сообщения об исключительных ситуациях, информация в окне программы соответствует его внутреннему состоянию.

Пользовательский сценарий 1. Подключение к FTP-серверу.
 Действующее лицо: Клиент
 Предусловие: Пользователь находится на главном экране программы, видна панель подключения.
 Основной сценарий:
  - Ввести адрес хоста
  - Ввести имя пользователя
  - Ввести пароль (не обязательно)
  - Ввести порт (не обязательно)
  - Нажать кнопку «Быстрое соединение»
 Альтернативный сценарий:
  - Нажать на кнопку со стрелочкой, рядом с кнопкой «Быстрое соединение»
  - Выбрать из списка сохранённый вариант авторизации
 Постусловие: вывод информации в лог о состоянии подключения; вывод информации о файлах на сервере в серверную панель (при успешном подключении).

Пользовательский сценарий 2. Передача файлов на FTP-сервер.
 Действующее лицо: Клиент
 Предусловие: Пользователь подключен к FTP-серверу и находится во вкладке с соответствующим сеансом.
 Основной сценарий:
  - В локальной панели перейти в каталог c файлами, которые нужно передать на сервер
  - В серверной панели перейти в каталог, в который нужно передать файлы
  - Выделить все необходимые файлы и папки в локальной панели
  - Мышкой перетащить файлы в серверную панель (либо нажать ПКМ -> «Загрузить»)
 Постусловие: файлы доставлены на сервер и отображены в серверной панели, либо выведено сообщение об ошибке.

Пользовательский сценарий 3. Загрузка файлов с FTP-сервера.
 Действующее лицо: Клиент
 Предусловие: Пользователь подключен к FTP-серверу и находится во вкладке с соответствующим сеансом.
 Основной сценарий:
  - В серверной панели перейти в каталог с файлами, которые нужно скачать
  - В локальной панели перейти в каталог, в который нужно скачать файлы с сервера
  - Выделить все необходимые файлы и папки в серверной панели
  - Мышкой перетащить файлы в локальную панель (либо нажать ПКМ -> «Скачать»)
 Постусловие: файлы доставлены на компьютер и отображены в локальной панели, либо выведено сообщение об ошибке.


Пользовательский сценарий 4. Поиск файлов на FTP-сервере.
 Действующее лицо: Клиент
 Предусловие: ППользователь подключен к FTP-серверу и находится во вкладке с соответствующим сеансом.
 Основной сценарий:
  - Выбрать пункт меню «Сервер» -> Поиск файлов на сервере
  - Выбрать тип поиска
  - Указать начальный каталог для поиска
  - Указать условия для поиска
  - Нажать кнопку «Искать»
 Постусловие: найденные файлы отображены в разделе «Результаты»


Пользовательский сценарий 5. Настройка прав доступа.
 Действующее лицо: Клиент
 Предусловие: Пользователь подключен к FTP-серверу и находится во вкладке с соответствующим сеансом.
 Основной сценарий:
  - Выбрать нужные файлы или каталоги
  - Кликнуть правой кнопкой мыши по выделенным объектам, выбрать пункт «Права доступа к файлу»
  - Указать каталог для поиска
  - Задать нужные права
  - Нажать кнопку «ОК»
 Постусловие: в лог выведено сообщение об изменении прав доступа на файл(ы)
