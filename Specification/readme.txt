Задание 1: написать спецификацию в виде сценариев пользователя.
Инварианты: Главное окно, менеджер звуковых эффектов, окно для плейлиста, панель управления треком, громкость.
Пользовательский сценарий 1: Добавление песни или плейлиста для прослушивания
Действующее лицо: Пользователь
Предусловие: Приложение запущено, песня (плейлист) скачан на компьютер
Основной сценарий:
1)	Выбор песни (плейлиста) для прослушивания
2)	Загружаем песню (плейлист) в окно Downloads
3)	Нажимаем на кнопку запуска в панели управления треком (пробел по умолчанию)
4)	При желании можно поставить на паузу, нажав на кнопку в панели управления
5)	При желании можно переключить на следующий трек самостоятельно, нажав соответствующую кнопку в панели управления (аналогично, можно и включить предыдущий) 
6)	Можно остановить прослушивание песни (плейлиста) полностью, нажав на “стоп” в панели управления (при дальнейшем запуске песни или плейлист будет запущен с самого начала)
7)	Можно удалить файл из плейлиста, нажав ПКМ на него и выбрав “Удалить”
Постусловие: в конце прослушивания песни (плейлиста), AIMP закончит воспроизведение. Можно включить функцию повтор файла на панели управления, чтобы плеер не останавливал воспроизведение. 
Пользовательский сценарий 2: Обработка звука  с помощью эквалайзера
Действующее лицо: Пользователь
Предусловие: Приложение запущено 
Основной сценарий:
1)	Заходим в менеджер звуковых эффектов
2)	Выбираем категорию “эквалайзер” (Рис. 2)
3)	При желании можно выбрать существующую преднастройку эквалайзера в разделе “преднастройки” 
4)	При желании можно настроить эквалайзер вручную, поднимая и опуская ползунки частот
5)	При желании откатить к начальным настройкам, есть кнопка “По умолчанию”
6)	Можно включить и выключить эквалайзер с помощью галочки
7)	При нажатии ПКМ на любой ползунок, он откатывается к значению по умолчанию
Постусловие: изменение звучания файла
Пользовательский сценарий 3: Использование звуковых эффектов
Действующее лицо: Пользователь
Предусловие: Приложение запущено
Основной сценарий: 
1)	Заходим в менеджер звуковых эффектов
2)	Выбираем категорию “Звуковые эффекты”
3)	При желании добавить какой-либо эффект, нужно передвинуть соответствующий ползунок вправо 
4)	При желании увеличить скорость файла, можно установить нужную скорость с помощью ползунка
5)	При желании увеличить темп файла отдельно от тональности, нужно передвинуть соответствующие ползунки
6)	Можно подавить голос в песне с помощью галочки в нижней панели 
7)	Можно установить баланс L и R с помощью ползунка
8)	При нажатии ПКМ на любой ползунок, он откатывается к значению по умолчанию
Постусловие: изменение звучания файла
