Задание 1: написать спецификацию в виде сценариев пользователя.
Инварианты: Главное окно, менеджер звуковых эффектов, окно для плейлиста, панель управления треком, громкость.
Пользовательский сценарий 1: Загрузка и прослушивание нескольких файлов
Действующее лицо: Пользователь
Предусловие: Приложение запущено, файлы для тестирования скачаны на компьютер
Основной сценарий: 
1)	Выбираем 2 файла для прослушивания
2)	Загружаем файлы в окно Downloads
3)	Нажимаем на кнопку запуска в панели управления треком (пробел по умолчанию)
4)	Ставим на паузу, нажав на кнопку в панели управления
5)	Переключаем на следующий файл, нажав соответствующую кнопку в панели управления 
6)	Останавливаем прослушивание файлов полностью, нажав на “стоп” в панели управления 
7)	Удаляем файлы из плейлиста, нажав ПКМ на него и выбрав “Удалить”
Постусловие: После удаления файлов, окно Downloads должно остаться пустым
Пользовательский сценарий 2: Обработка звука  с помощью эквалайзера
Действующее лицо: Пользователь
Предусловие: Приложение запущено, файлы для тестирования скачаны на компьютер и загружены в плеер

Основной сценарий:
1)	Заходим в менеджер звуковых эффектов
2)	Выбираем категорию “эквалайзер” (Рис. 2)
3)	Выбираем существующую преднастройку эквалайзера в разделе “преднастройки” под названием Club
4)	Настраиваем эквалайзер вручную, подняв ползунок частоты 8к гц до 3дб
5)	Включаем эквалайзер, если он не был включен с помощью галочки
6)	Нажимаем ПКМ на ползунок обозначающий частоту 250гц, он откатывается к значению по умолчанию
Постусловие: файл изменил звучание. В эквалайзере ползунки установлены по преднастройке “Club” кроме двух ползунков – 8к гц установлен на 3дб, а 250 гц установлен на 0 дб (по умолчанию).
Пользовательский сценарий 3: Использование звуковых эффектов
Действующее лицо: Пользователь
Предусловие: Приложение запущено, файлы для тестирования скачаны на компьютер и загружены в плеер
Основной сценарий: 
1)	Заходим в менеджер звуковых эффектов
2)	Выбираем категорию “Звуковые эффекты”
3)	Добавляем эффект “Бас” до значения 1.00, передвинув соответствующий ползунок вправо 
4)	Увеличиваем скорость файла до значения 1.10, передвинув соответствующий ползунок вправо 
5)	Уменьшаем темп файла до 0.90, передвинув соответствующий ползунок влево
6)	Устанавливаем баланс L и R с помощью ползунка, полностью передвинув его вправо, чтобы файл звучал только в справа
Постусловие: изменение звучания файла. Файл звучит только справа, у него повысилась тональность, но в темпе он остался таким же, каким был до добавления эффектов, бас стало лучше слышно.
